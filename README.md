# UnifCore

A  modular business management platform powered by plugins.

UnifCore is built around a simple idea:

> Install only what you need.

Instead of shipping a all-in-one ERP, UnifCore provides a minimal core system with installable business modules such as HR, Inventory, POS, School Management, CRM, and more.

---

# Features

- Plugin-based architecture
- Organization & multi-tenant support
- Role-based access control (RBAC)
- Event-driven module communication
- Extensible API-first backend
- Runtime sidebar & route injection
- Scalable business application platform

---

# Philosophy

Traditional ERP systems become bloated because every feature is enabled by default into the core.

UnifCore takes a different approach.

The core only contains universal platform features:

- Authentication
- Organizations
- Permissions
- Plugin Runtime
- Notifications
- Event System
- Settings
- Storage

Everything else is a plugin.


---

# Goals

- Keep the core minimal
- Make every feature pluggable
- Support third-party plugins
- Build a scalable business app ecosystem
